# Project Images

This directory should contain the following images for the project cards:

## Required Images

1. **tidytuesday.png** - TidyTuesday project thumbnail
   - Recommended size: 400x300px or similar aspect ratio
   - Should show a sample TidyTuesday visualization

2. **delgosha.png** - Delgosha R package thumbnail
   - Recommended size: 400x300px or similar aspect ratio
   - Could show Persian text visualization or package logo

3. **palmer.png** - Palmer Penguins package thumbnail
   - Recommended size: 400x300px or similar aspect ratio
   - Could show penguin illustrations or sample visualization

## Notes

- Images are referenced in the respective project index.qmd files
- If images are missing, Quarto will still render but show placeholder or no image
- You can copy these images from your old Distill blog's `imgs/` directory
- Supported formats: PNG, JPG, SVG

## Image Locations

- TidyTuesday: `projects/projects/tidytuesday/tidytuesday.png`
- Delgosha: `projects/projects/delgosha/delgosha.png`
- Palmer Penguins: `projects/projects/palmerpenguins/palmer.png`
